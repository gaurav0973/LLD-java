# Core principle of OOP
They are called the four pillars of OOP. They are:
1. Encapsulation
2. Abstraction
3. Inheritance
4. Polymorphism



## Encapsulation
- What is Encapsulation?
    - Wrapping data (variables) and code (methods) together as a single unit.   
- Why Encapsulation?
    - Data Hiding
    - Data protection
    - Controlled access
    - Maintainability
- How to achieve Encapsulation in Java?
    - Access Modifiers
        - Private
        - Default
        - Protected
        - Public
    - Getters and Setters
